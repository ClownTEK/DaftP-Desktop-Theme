# DaftP
Desktop theme for Fluxbox containing:
* GTK2
* Fluxbox theme
* Fluxbox init
* Icons

By pobtott: http://pobtott.deviantart.com/art/Any-Color-You-Like-175624910.
* Geany colorscheme
* Xdefaults
* Dircolors
* ls++.conf
* bashrc
* Wallpaper

Link: https://awesomewallpapersblog.com/2016/11/04/abstract-wallpaper-set-93/.
